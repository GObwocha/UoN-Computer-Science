# 🌍 NEMA Environmental Monitoring System (Project 34)
## 📌 Project Overview
This repository contains the database system for Project 34 – Environmental Monitoring, designed to streamline and digitalize the operations of the National Environment Monitoring Agency (NEMA).

Developed as a comprehensive Computer Science database project, this system provides a robust backend and an intuitive, custom-styled frontend to manage environmental experts, licensing, audits, and incident reporting.

## ✨ Key Features
🔐 Secure Authentication: Custom-built, modal login and registration screens completely uncoupled from underlying tables, utilizing VBA for credential validation and secure access.

📊 Dashboard Navigation: A centralized, tabbed navigation interface ("Forest Canopy" theme) providing seamless access to all system modules without exposing the raw Access ribbon or navigation pane.

🚨 Incident Management: Comprehensive tracking of environmental emergencies, featuring categorized dropdowns for specific incidents such as:

Illegal Effluent Discharge, Toxic Gas/Chemical Leaks, Riparian Encroachment, Dumpsite Fires, Agrochemical Runoff, and more.

📄 Automated Reporting: Dynamically generated, auto-centering reports that maximize to fit the screen, maintaining high-quality visual formatting and custom UI color palettes.

⚙️ Data Integrity: Utilization of strict validation rules, input masks, and unbound forms to ensure accurate data entry and prevent accidental table modification.

## 🗂️ Repository Contents

## 🛠️ Technical Stack
RDBMS: Microsoft Access

Programmability: Visual Basic for Applications (VBA)

Design & UI: Custom Microsoft Access Form controls, integrated Shape Effects, and custom Color Themes (Hex/RGB overrides).

Architecture: Relational Database Design (UML ERD)
